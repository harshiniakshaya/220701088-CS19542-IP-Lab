jQuery Stock Functions Examples

1. $.each()

This function is used to iterate over arrays or objects.


const fruits = ["Apple", "Banana", "Cherry"];

$.each(fruits, function(index, value) {
  console.log(index + ": " + value); // Logs each fruit with its index
});

2. $.isArray()

This function checks if a value is an array.


const items = [1, 2, 3];
const notArray = "Hello";

console.log($.isArray(items)); // true
console.log($.isArray(notArray)); // false

3. $.trim()

This function removes whitespace from the beginning and end of a string. 


const stringWithSpaces = "  Hello World!  ";
const trimmedString = $.trim(stringWithSpaces);

console.log(trimmedString); // Logs "Hello World!" without surrounding spaces

4. $.extend()

This function merges the contents of two or more objects into the first object. 


const defaults = { name: "Unknown", age: 0 };
const person = { name: "Alice", age: 25, place:"chennai"  };

const extended = $.extend({}, defaults, person);

console.log(extended); // Logs { name: "Alice", age: 25 }

5. $.type()

This function returns the type of the argument passed.


console.log($.type(123)); // "number"
console.log($.type("Hello")); // "string"
console.log($.type([1, 2, 3])); // "array"
console.log($.type({ key: "value" })); // "object"

6. $.inArray()

This function checks if a value is in an array and returns its index; otherwise, it returns -1. 


const colors = ["red", "green", "blue"];

console.log($.inArray("green", colors)); // 1
console.log($.inArray("yellow", colors)); // -1

7. $.isEmptyObject()

This function checks if an object is empty (has no properties).


const emptyObject = {};
const nonEmptyObject = { name: "John" };

console.log($.isEmptyObject(emptyObject)); // true
console.log($.isEmptyObject(nonEmptyObject)); // false

8. $.now()

This function returns the current timestamp (milliseconds since January 1, 1970). 


console.log($.now()); // Logs the current timestamp

9. $.map()

This function applies a function to each item in an array and returns a new array with the results. 


const numbers = [1, 2, 3];
const squares = $.map(numbers, function(num) {
  return num * num;
});

console.log(squares); // [1, 4, 9]

10. $.proxy()

This function changes the context of this inside a function, which is useful when passing functions as arguments. 


const person = {
  name: "Alice",
  sayName: function() {
    console.log(this.name);
  }
};

const button = $("#myButton");
button.on("click", $.proxy(person.sayName, person));
